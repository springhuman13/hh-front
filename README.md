# 💻 HaHackathon Frontend

Frontend for the Hackathon Hub platform.

The application provides an interface for finding hackathons, creating teams, managing profiles and interacting with the backend API.

---

## Features

- Telegram authentication
- User profiles
- Team creation
- Team browsing
- Skill matching
- Document upload
- Responsive interface

---

## Tech Stack

- HTML5
- CSS3
- JavaScript

---

## Backend

The frontend communicates with the FastAPI backend via REST API.

---

## Getting Started

Run with Live Server or

```bash
python -m http.server
```
